Estandares X.800 y RFC 4949

> ITU-T > IETF > Estandares

# X.800 - Arquitectura de Seguridad OSI

El estandar X.800 de ITU-T define la arquitectura de seguridad para el modelo OSI. Establece un marco de referencia para implementar servicios y mecanismos de seguridad en sistemas de comunicacion.

# Servicios de Seguridad (X.800)

  • Autenticacion: Verificacion de identidad de entidades
  • Control de Acceso: Prevencion de uso no autorizado
  • Confidencialidad: Proteccion contra divulgacion no autorizada
  • Integridad: Deteccion de modificaciones no autorizadas
  • No Repudio: Proteccion contra negacion de acciones

# RFC 4949 - Internet Security Glossary

El RFC 4949 es un glosario que define terminologia de seguridad utilizada en documentos de Internet. Proporciona definiciones precisas para conceptos clave en ciberseguridad.

# Terminos Clave (RFC 4949)

$ cat /etc/security/glossary.conf

THREAT="Potencial violacion de seguridad"
VULNERABILITY="Debilidad en un sistema"
ATTACK="Asalto a la seguridad del sistema"
RISK="Probabilidad de amenaza explotando vulnerabilidad"
COUNTERMEASURE="Accion para reducir amenaza/vulnerabilidad"
ASSET="Recurso del sistema con valor"

# Comparacion de Estandares

  • X.800: Enfoque en arquitectura y servicios de seguridad
  • RFC 4949: Enfoque en terminologia y definiciones
  • Ambos son complementarios para entender seguridad de redes
  • X.800 es mas tecnico/implementacion, RFC 4949 es mas conceptual
