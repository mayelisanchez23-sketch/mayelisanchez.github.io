# Defensa en Profundidad

La defensa en profundidad es una estrategia que utiliza multiples capas de seguridad para proteger los activos de una organizacion. Si una capa falla, las siguientes continuan proporcionando proteccion.

# Capas de Defensa

CAPAS_SEGURIDAD = [
  "1. Perimetro (Firewall, DMZ)",
  "2. Red (IDS/IPS, Segmentacion)",
  "3. Host (Antivirus, HIDS)",
  "4. Aplicacion (WAF, Validacion)",
  "5. Datos (Cifrado, DLP)",
  "6. Usuario (Capacitacion, MFA)"
]

# Sistemas IDS/IPS

  • IDS (Intrusion Detection System): Detecta y alerta sobre actividad sospechosa
  • IPS (Intrusion Prevention System): Detecta y bloquea automaticamente
  • NIDS: Monitorea trafico de red
  • HIDS: Monitorea actividad en hosts individuales
