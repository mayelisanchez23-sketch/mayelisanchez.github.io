ACT_06 - Parcial 2

IPSec VPN Configuration

> IPSec > Site-to-Site > Tunneling

# Que es IPSec?

IPSec (Internet Protocol Security) es un conjunto de protocolos que proporcionan seguridad a nivel de red mediante autenticacion y cifrado de paquetes IP. Se utiliza principalmente para crear VPNs seguras.

# Componentes de IPSec

  • AH (Authentication Header): Integridad y autenticacion
  • ESP (Encapsulating Security Payload): Confidencialidad, integridad y autenticacion
  • IKE (Internet Key Exchange): Negociacion de claves
  • SA (Security Association): Parametros de seguridad acordados

# Modos de Operacion

# Modo Transporte
- Solo cifra el payload
- Header IP original se mantiene
- Usado para comunicacion host-to-host

# Modo Tunel
- Cifra todo el paquete IP original
- Nuevo header IP se agrega
- Usado para VPN site-to-site

# Configuracion Site-to-Site

# Fase 1 - IKE (ISAKMP)
crypto isakmp policy 10
 encryption aes 256
 hash sha256
 authentication pre-share
 group 14
 lifetime 86400

crypto isakmp key CLAVE_SECRETA address 203.0.113.1

# Fase 2 - IPSec Transform Set
crypto ipsec transform-set TSET esp-aes 256 esp-sha256-hmac
 mode tunnel

# Crypto Map
crypto map VPN_MAP 10 ipsec-isakmp
 set peer 203.0.113.1
 set transform-set TSET
 match address VPN_ACL

# Verificacion

# Verificar estado de IKE
$ show crypto isakmp sa

# Verificar tuneles IPSec
$ show crypto ipsec sa

# Ver estadisticas
$ show crypto engine connections active
