> Detalle tecnico del modulo

Esta seccion resume el enfoque metodologico, los tipos de SQLi abordados y la evidencia academica integrada en la adaptacion web del proyecto.

> Metodologia aplicada
Burp Suite Web Security Academy

Los laboratorios fueron estructurados a partir de la identificacion del punto vulnerable, la manipulacion de peticiones HTTP, la construccion de payloads y la verificacion de evidencia tecnica mediante respuestas, mensajes, cambios de comportamiento o extraccion de informacion.

  • [+]Interceptacion y modificacion de requests.
  • [+]Pruebas con payloads UNION, error-based, blind y out-of-band.
  • [+]Validacion del resultado con evidencia y estado "Lab Solved".
  • [+]Documentacion resumida orientada a portafolio profesional.
> Tipos de SQL Injection incluidos
Cobertura completa

El modulo integra distintos escenarios de explotacion para mostrar la evolucion tecnica del tema, desde bypass de autenticacion hasta exfiltracion condicionada y evasion de filtros.

  • >WHERE clause manipulation.
  • >Login bypass.
  • >UNION-based SQL Injection.
  • >Error-based SQL Injection.
  • >Blind SQL Injection: conditional responses / errors / time delays.
  • >Out-of-band y bypass de filtros via XML encoding.
